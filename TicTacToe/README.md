## About:
**TicTacToe** is a game app that represent an instance of the popular Tic Tac Toe game or NOUGHTS AND CROSSES.

**TicTacToe** is a Swift App that shows the implementation of:
- Sound integration
- Haptic feedback integration
- AutoLayout
