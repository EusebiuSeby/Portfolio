# ToDo App

![](./images/TD_logo.png)
## About:
**ToDo** is an app that provides you a to-do list where you can write down all your tasks that are waiting for completion.

**ToDo** is a Swift App that shows the implementation of:
- CoreData (CRUD)
- Custom TableView
- MVC Pattern
- AutoLayout
- Delegates and Protocols
